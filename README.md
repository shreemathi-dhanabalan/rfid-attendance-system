# RFID Smart Attendance System using NodeMCU

## Description
This project is an IoT-based Smart Attendance System using RFID and NodeMCU. It automatically records attendance and stores the data in an Excel file.

## How It Works
- Each user has an RFID card
- RFID module reads the card UID
- NodeMCU processes the data
- Attendance is recorded and stored in Excel

## Components Used
- NodeMCU (ESP8266)
- RFID Module (RC522)
- RFID Tags
- Jumper Wires

## Features
- Automatic attendance marking
- Real-time data processing
- Data stored in Excel
- Reduces manual errors

## Technologies Used
- Embedded C (Arduino IDE)
- IoT (NodeMCU)
- Serial/Communication for data logging

## Applications
- Colleges
- Schools
- Offices
